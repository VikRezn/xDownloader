# X Media Downloader
A Chrome extension that adds a button to download original quality photos and videos directly from X (Twitter).

![Download Button UI](https://github.com/user-attachments/assets/f260b795-5568-404a-887d-bec0206b7b79)
![Media Picker UI](https://github.com/user-attachments/assets/b256b13f-7f90-4e16-aad1-01815bc1127c)

## Features
* **Native Integration:** Injects a download button directly into the tweet action bar.
* **High Quality:** Downloads the highest resolution images (`name=orig`) and the highest bitrate MP4 videos.
* **Media Picker:** Displays a visual selection grid for tweets containing multiple media items.
* **Download Settings:** Click the extension icon to customize download behavior:
    * Download straight to the default Downloads folder.
    * Download to a specific subfolder within Downloads.
    * Prompt with a "Save As" dialog for every file.

## Installation
1. Download and extract this repository to your computer.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Toggle on **Developer mode** in the top right corner.
4. Click **Load unpacked** in the top left.
5. Select the extracted folder containing the extension files.
